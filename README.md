# Alarm Clock

A simple web-based alarm clock application built using HTML, CSS, and JavaScript. You can set alarms and be notified when the alarm time is reached.

## Features

- Set multiple alarms with a specified time.
- Turn off or remove alarms.
- Alarm sound notification.
- Responsive design for various screen sizes.

## Demo

You can try out the alarm clock application in action by visiting the [live demo](https://thewebalpha-alarmclock.netlify.app/).

## Getting Started

### Prerequisites

You need a modern web browser to run this application. No additional dependencies or installations are required.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Omotola-ni/Alarm-Clock.git
   ```

2. Open the `index.html` file in your web browser.

## Usage

1. Set Alarm:
   - Enter the desired alarm time in the input field.
   - Click the "Set Alarm" button to schedule the alarm.

2. Turn Off Alarm:
   - When an alarm goes off, a notification sound will play.
   - You can turn off the alarm by clicking the "Turn Off" button.

3. Remove Alarm:
   - You can remove an alarm by clicking the "Remove" button.
